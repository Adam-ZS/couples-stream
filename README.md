# couples.stream

Watch movies together in real-time sync. Share a link, both watch, zero setup.

## How it works

1. Open the site → click **Create Room**
2. Share the generated link with your partner
3. Search any movie, pick a source
4. Both watch in perfect sync with built-in chat

No accounts. No signup. No downloads. Just the link.

## Features

- Room creation with instant shareable link
- Movie search via TMDB
- Auto-fetches streams from Torrentio + TPB+ + HiTV
- Real-time play/pause/seek sync via WebSocket
- Built-in room chat
- Server-side torrent streaming (no browser plugins)
- Mobile-friendly dark UI
